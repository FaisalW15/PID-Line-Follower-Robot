# PID-Line-Follower-Robot
This is my attempt at programming and building a wheeled robot that follows a line as it twists and curves using a programmed PID controller.
The motivation behind this project is to understand real world applications of the PID controller, and to test adjusting each parameter (Proprtional, Integral and Derivative)
to get a sense of how increasing and decreasing each gain parameter effects the error correction.
